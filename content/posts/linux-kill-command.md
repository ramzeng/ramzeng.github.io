## 常用
- SIGHUP(1)：挂起信号
- SIGINT(2)：中断信号
- SIGQUIT(3)：退出信号
- SIGKILL(9)：强制终止信号
- SIGTERM(15)：终止信号
- SIGSTOP(19)：停止信号
- SIGCONT(18)：继续信号

## 命令
```bash
# 默认发送 SIGTERM 信号
kill {PID}

# 发送 SIGKILL 信号
kill -9 {PID}

# 重启 PHP-FPM
kill -USR2 {PID}
kill -12 {PID}

# 重启 Nginx
kill -HUP {PID}
kill -1 {PID}
```
