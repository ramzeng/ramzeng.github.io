## Components
- [Hugo](https://gohugo.io)
- [Hugo-Paper](https://themes.gohugo.io/themes/hugo-paper)

## How to use
Install Hugo
```bash
brew install hugo
```
Clone this repository
```bash
git clone github.com/ramzeng/ramzeng.github.io 
```
Run hugo server
```bash
cd ramzeng.github.io && hugo server -D
```
